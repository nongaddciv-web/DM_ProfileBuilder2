Create a Profile from a Face

Draw a Face with SketchUp

1. Draw any face in SketchUp at any location.
2. Select the Face.

Open the Profile Builder Dialog

Click the 'New Profile' Button

Give the Profile a Name

1. Type the name of the Profile and Click OK
2. All profiles must be given a name.

Congratulations! You just made a Profile!
