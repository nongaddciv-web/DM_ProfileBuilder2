Managing Profiles

Editing Saved Profiles

Since profiles are stored as SKP files, to edit them, just open the SKP file, make the changes and then re-save.

Tip: Be sure that the SKP only has one face or polyline and it is not inside any group or component.

Organizing Profiles

Instead of using Profile Library Files (PLB files) like in the original Profile Builder, your profiles should be stored in an organized system of folders on your local computer or server, similar to how you might store your SketchUp Component Library.

Profile folders can be stored at any location as long as you have security permissions to write files to that particular folder.
